## Key Features of this App:
Modern UI: 
Uses CustomTkinter for rounded corners, smooth hover effects, and a dark theme by default.

Real-time Data: 
Fetches data from OpenWeatherMap, including temperature, humidity, and wind speed.

Dynamic Icons:
It downloads the weather icon (sun, clouds, rain) directly from the API and displays it in the app.

Responsive:
Elements are centered and padded for a clean look.

## How to use it:
Go to OpenWeatherMap.
Create a free account and generate an API Key.
Paste your key into the API_KEY variable inside the script.
## Run the script:
python weather_app.py.
